# Skill Creator Meta-Skill
# Goal: Help zmariscal create new .claude skills.

## 📝 Format Requirements
- File must be named `SKILL.md`.
- Use a clear H1 title and a "Goal" or "Philosophy" section.
- Use H2 for categories (e.g., ## Database, ## Testing).
- Use bullet points for actionable rules.

## 🛠 Placement
- Always suggest placing new skills in `~/dotfiles/files/home/.claude/skills/[skill-name]/`.
- Remind the user to update `AGENTS.md` and run `scripts/setup` after creation.
